### How to run this?

- Clone on your local machine

```terminal
git clone https://github.com/Spyware007/Mentro-Assignment.git
```

- Change directory to Mentro-Assignment

```terminal
cd Mentro-Assignment
```

- Install all npm dependencies/packages

```terminal
npm i
```

- Run the project

```
npm start
```
